## 게임 서버

아주 간단한 게임 서버를 생각해봅시다. 아래처럼 클라이언트와 서버가 아래와 같이 연결되어 있습니다.

{{<mermaid>}}
flowchart LR
    클라이언트1 <--> 서버
    클라이언트2 <--> 서버
    클라이언트3 <--> 서버
{{</mermaid>}}

이처럼 다수의 클라이언트들이 서버를 통해서 상호작용합니다. 지금은 서버가 처리해야 할 클라이언트도 많지 않고, 사용자의 데이터를 영구적으로 저장하는 것도 고려되지 않았는데 하나씩 더 살펴봅시다.
